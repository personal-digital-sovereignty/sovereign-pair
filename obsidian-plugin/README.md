# Sovereign Pair - Obsidian Plugin

> **Your Universal Local AI Assistant. Connects Obsidian to your private RAG backend (Ollama/OpenAI) via Sovereign Pair.**

Sovereign Pair is a powerful, privacy-first Retrieval-Augmented Generation (RAG) system. This plugin is the official frontend client that injects the Sovereign Pair intelligence directly into your Obsidian Vault seamlessly.

## 🌟 Key Features

1. **True Intellectual Sovereignty:** Keep your notes private. No data is sent to third-party servers unless you explicitly configure a Cloud Provider (like OpenAI). By default, it connects to your local `Ollama` models.
2. **Three Advanced Search Modes:**
   - **🧠 Sovereign RAG (Vector Search):** Ask questions, and the AI will only answer based on the knowledge inside your Obsidian Vault.
   - **🌐 Web Search:** The AI browses DuckDuckGo live to answer questions about recent events, citing the sources.
   - **🤖 Pure LLM:** A standard, lightning-fast chat with your favorite models (Llama 3, Mistral, GPT-4o).
3. **Frictionless Ingestion:** Right-click any note or just paste a file into the chat to instantly vectorize it into your RAG brain.
4. **Mini-Web UI:** A sleek, fully responsive chat interface living right inside your right sidebar, complete with Message Editing, Session Folders, and AI Personas.

## ⚙️ How to Setup

This plugin requires the **Sovereign Pair Backend API** to be running. You can host this backend anywhere: on your local Gamer PC, on a Raspberry Pi, or on a Cloud VPS (like Oracle Cloud).

1. Install this plugin from the Community Plugins tab.
2. Enable the plugin. A **Welcome Setup** modal will appear.
3. Determine your **API Base URL** (e.g., `https://localhost` if running locally, or `https://rag.yourdomain.com`).
4. Paste your **Sovereign Security Token** (this is generated by your backend to prevent unauthorized access to your vector database).
5. Click **Save and Connect**.

*If you haven't set up the Python Backend yet, don't worry! It takes less than 2 minutes using Docker.*
**👉 [Get the Backend Server on our GitHub](https://github.com/Personal-Digital-Sovereignty/sovereign-pair)**

## 🛠️ For Developers: Build Your Own Backend!

Don't want to use our Python FastAPI backend? No problem! This plugin was built explicitly to be **Agnostic**.
You can write your own backend in Node.js, Go, or Rust, and this plugin will connect to it perfectly, so long as your API respects our JSON schema. 

Read the [Sovereign Pair API Contract (API_CONTRACT.md)](https://github.com/Personal-Digital-Sovereignty/sovereign-pair/blob/main/obsidian-plugin/API_CONTRACT.md) to learn how to interface with this UI!
